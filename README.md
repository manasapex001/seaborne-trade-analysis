# seaborne-trade-analysis
This project analyses global seaborne trade volumes across four major commodity segments — Crude Oil &amp; Tankers, Dry Bulk, Containers, and LNG — from 2010 to 2023. It tracks structural shifts in tonne-mile demand, models LNG growth using linear regression, and estimates the tonne-mile demand impact of the 2023 Red Sea disruption on Asia–Europe .
